# E-Commerce Order Management Database System

## Week 1 – Business Requirement Analysis

### Project Overview

This project focuses on understanding the business requirements of an **E-Commerce Order Management Database System**.
The objective is to analyze the business processes, identify system requirements, and prepare a **Software Requirement Specification (SRS)** document that serves as the foundation for future database design and implementation.

## Project Objectives

* Analyze the business requirements of an e-commerce system.
* Understand the core entities involved.
* Identify business processes and stakeholders.
* Define functional and non-functional requirements.
* Prepare a Software Requirement Specification (SRS).
* Define project scope, assumptions, and constraints.
